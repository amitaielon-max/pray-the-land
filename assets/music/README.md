# Music for the films

Only tracks Tai owns or licenses go in this folder. The editing room uses
nothing else. Audio files here are gitignored on purpose: they stay local
to the editing session and are never deployed with the site.

If this folder is empty, films render without music, which is fine.

What works under these films: quiet acoustic guitar or strings, slow,
no percussion-heavy beds. The music sits far under the voice (about
-26 LUFS, ducked when Tai speaks) and stays silent during the Scripture
turn unless Tai asks otherwise.
